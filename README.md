# DoitNad
DoitNad is rock, paper, scissors game play with bot.The game build on Monad testnet.

![doitnad](./doitnad.png)

# How to play
- Connect wallet

- Choice Rock, paper, scissors

- Click button "Play"

- Confirm transaction

- Get your point
